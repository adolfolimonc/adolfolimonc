## 📦 Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/yourusername/adolfolimonc.git
   cd adolfolimonc
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or
   yarn install
   ```

3. **Run the development server:**
   ```bash
   npm run dev
   # or
   yarn dev
   ```

4. **Open [http://localhost:3000](http://localhost:3000) to view it in your browser.**

---

## 📝 Customization

- **Projects:** Add or edit your projects in `/data/projects.json`
- **Content:** Update text and images in the `/data` and `/public/assets` folders
- **Styling:** Modify styles in `/public/assets/css` or use Tailwind classes
